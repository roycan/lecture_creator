🧰 Environment Setup Checklist

Get your laptop ready to build & run a Node/Express app. Do this once at the start of the project. Print it and tick each step.

  1. Install Node.js (includes npm) from nodejs.org. Pick the "LTS" version.
  2. Verify the install — open a terminal and run: 
    node -v
npm -v
    You should see two version numbers. If not, reopen the terminal or reinstall.
  3. Create your project folder and go into it: 
    mkdir my-app
cd my-app
  4. Create package.json (accept the defaults): 
    npm init -y
  5. Install Express: 
    npm install express
    This creates node_modules/ and saves Express to package.json.
  6. Create your secret config file — .env: 
    STORE_NAME=Practice Store
ADMIN_PASSWORD=Practice123!
PORT=3000
    (Never put real passwords in code — read them from process.env instead.)
  7. Create .gitignore so secrets & heavy folders are NOT pushed to GitHub: 
    node_modules/
.env
  8. Create app.js with a tiny Express server, then run it: 
    node app.js
  9. Open it at http://localhost:3000. You should see your page. 🎉
  10. To stop the server: click the terminal and press Ctrl + C.
💡 Two golden rules:
  • .env holds secrets — it's in .gitignore and never committed to GitHub.
  • node_modules/ is huge and rebuilt with npm install — never copy it by USB or commit it.
