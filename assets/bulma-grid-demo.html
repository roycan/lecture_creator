üí° Grid System Tips

  1. Start with mobile: Design for small screens first, then expand
  2. Use is-mobile for icon grids: Small elements look better side-by-side even on phones
  3. Main content + sidebar: Use is-8 + is-4 or is-9 + is-3
  4. Product grids: Use is-half-mobile is-one-third-tablet is-one-quarter-desktop
  5. Centered forms: Use is-centered with is-half or is-two-thirds
  6. Always test: Resize your browser window to see how columns adapt!
