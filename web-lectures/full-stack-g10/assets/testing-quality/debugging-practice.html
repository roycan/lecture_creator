A simple calculator… with 5 bugs. Open DevTools (F12) and use the Console + Elements to find and fix them.

💡 How to debug: click Calculate, read the red error in the Console, click the file:line to jump to the code, then inspect the Elements tab to check the real id names.
🔓 Show the 5 bugs (teacher / check-yourself)
  1. Wrong element id: getElementById('prce') should be 'price'. Causes TypeError: Cannot read properties of null.
  2. String math: price + qty joins text ("502") instead of multiplying numbers. Use parseFloat().
  3. Discount threshold: subtotal > 500 skips a subtotal of exactly 500. Should be >= 500.
  4. Result written to wrong id: getElementById('totl') should be 'result'.
  5. Clear button: onclick="clearForm()" but the function is resetForm(). Causes clearForm is not defined.
