Image Best Practices

  • Always use alt text - Describes image for screen readers
  • Use descriptive filenames - barangay-hall.jpg not img001.jpg
  • Optimize file size - Large images slow down websites
  • Use appropriate formats - JPG for photos, PNG for logos
  • Make images responsive - Use max-width: 100% for mobile
