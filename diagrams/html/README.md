## Overview

This directory contains visual diagrams for the **HTML Fundamentals Lecture**. These diagrams help Grade 9 students understand HTML document structure, semantic tags, forms, tables, links, and attributes through visual representations.

---

## Diagram Inventory

### 1. **Document Structure** (`01-document-structure.mmd`)
- **Format:** Mermaid flowchart
- **Purpose:** Shows HTML document tree structure
- **Content:**
  - DOCTYPE declaration
  - html → head + body hierarchy
  - Meta tags in head
  - Semantic sections in body
- **Key Concepts:** Document structure, head vs body, semantic HTML5

### 2. **Semantic Layout** (`02-semantic-layout.txt`)
- **Format:** ASCII art
- **Purpose:** Visual layout of semantic HTML5 elements
- **Content:**
  - header (navigation, title)
  - main (sections, articles)
  - aside (sidebar)
  - footer (copyright, contact)
- **Key Concepts:** Semantic HTML5, page structure, meaningful tags

### 3. **Heading Hierarchy** (`03-heading-hierarchy.txt`)
- **Format:** ASCII tree diagram
- **Purpose:** Shows proper heading structure (h1-h6)
- **Content:**
  - Tree structure of headings
  - Real example: School website
  - Rules for heading order
  - Common mistakes
- **Key Concepts:** Heading levels, hierarchy, accessibility, SEO

### 4. **Form Structure** (`04-form-structure.txt`)
- **Format:** ASCII tree + examples
- **Purpose:** Demonstrates form organization and label association
- **Content:**
  - fieldset + legend grouping
  - label + input association (for/id)
  - Common input types
  - Student registration example
- **Key Concepts:** Forms, labels, accessibility, input types

### 5. **Table Structure** (`05-table-structure.txt`)
- **Format:** ASCII tables + code
- **Purpose:** Shows table anatomy and rowspan/colspan
- **Content:**
  - thead, tbody, tfoot sections
  - th vs td cells
  - rowspan and colspan examples
  - Student grade sheet example
- **Key Concepts:** Tables, data presentation, spans, accessibility

### 6. **Link Types** (`06-link-types.txt`)
- **Format:** Code examples + explanations
- **Purpose:** Explains different link types and uses
- **Content:**
  - Internal links (relative paths)
  - External links (absolute URLs)
  - Anchor links (#section)
  - Email links (mailto:)
  - Phone links (tel:)
  - School navigation example
- **Key Concepts:** Hyperlinks, navigation, paths, accessibility

### 7. **Attribute Syntax** (`07-attribute-syntax.txt`)
- **Format:** Annotated code + reference
- **Purpose:** Breaks down HTML attribute syntax
- **Content:**
  - Attribute anatomy
  - Global attributes (id, class, title, data-*)
  - Tag-specific attributes (href, src, type, etc.)
  - Boolean attributes (required, disabled)
  - Student form example
- **Key Concepts:** Attributes, values, quotes, syntax rules
