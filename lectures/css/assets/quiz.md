# Quiz: CSS Fundamentals

## Maria wants all the `<h1>` headings on her barangay website to be blue. Which CSS selector should she use?
* [ ] `.h1`
* [ ] `#h1`
* [x] `h1`
* [ ] `*h1`

## Carlo wants to center a `<div>` horizontally inside its parent container. He already set a fixed width. Which CSS property achieves this?
* [ ] `text-align: center`
* [x] `margin: 0 auto`
* [ ] `align: center`
* [ ] `position: center`

## Juan wrote this CSS, but the text color doesn't change. What is wrong?
```css
p {
  color; blue;
}
```
* [ ] `blue` should be in quotes
* [x] The separator between `color` and `blue` should be a colon `:` not a semicolon `;`
* [ ] `p` should be `.p`
* [ ] `color` is not a valid CSS property

## Liza wants to apply a style only to elements that have `class="highlight"`. Which selector syntax does she use?
* [ ] `#highlight`
* [x] `.highlight`
* [ ] `highlight`
* [ ] `[class=highlight]`

## The CSS ___ model treats every element as a rectangular box made of four layers: content, padding, border, and margin.
Answer: box model
Answer: box

## Ana has a button that looks too small on mobile. She wants to add 10 pixels of space *inside* the button, between the text and the border. Which property controls that inner space?
* [ ] `margin`
* [x] `padding`
* [ ] `spacing`
* [ ] `inner`

## Ben wants to display three product cards side by side on his sari-sari store page. Which CSS display value is most commonly used for this type of row layout?
* [ ] `display: inline`
* [x] `display: flex`
* [ ] `display: block`
* [ ] `display: table`

## This code is supposed to make the body background light gray, but nothing changes. What is the likely problem?
```css
Body {
  backround-color: #f5f5f5;
}
```
* [x] `backround-color` is misspelled — it should be `background-color`
* [ ] `Body` should be lowercase `body`
* [ ] Hex colors need quotes around them
* [ ] The property should be `bgcolor`
