Class What It Does When to Use
is-hidden-mobile Hides on phones (< 768px) Full navigation menus, detailed descriptions
is-hidden-tablet-only Hides only on tablets (769-1023px) Rarely used, specific tablet layouts

‚úÖ Best Practices

  1. Think Mobile First: Design for mobile, then enhance for desktop
  2. Keep Mobile Simple: Shorter text, fewer options, bigger buttons
  3. Test on Real Devices: Emulators are good, but real phones are better
  4. Save Data on Mobile: Use smaller images, fewer features
  5. Cards > Tables on Mobile: Always transform tables to cards
  6. Big Touch Targets: Buttons should be at least 44x44px
