Extending

  1. Edit data.js and add an object to TEMPLATES (key, label, category, engine, code).
  2. Use existing categories or introduce a new one (will appear automatically).
  3. Reload the app; the template will show when its engine is selected.

See logs/project-context.md for full details of the Template System.

Roadmap (Deferred)

  • BPMN – XML editing complexity (deferred).
  • Vega / Vega-Lite – Large client libs (bundle size concerns).
  • Excalidraw – Interactive canvas editor out of current scope.

Offline & Self-Hosting

Mermaid ⚡ and Plain Text 📝 engines work fully offline. Remote engines (🌐) require network access to Kroki. You can self-host Kroki locally (see guide) and set the base URL in the editor for offline-ish use.
