📖 How to use these lectures

Online: you're already here — just click any card below to open that lecture.

Offline (slow or no internet): download the whole folder as a ZIP, unzip it, and open this same index.html in any browser. Every lecture is a single self-contained HTML file — images, syntax highlighting, and diagrams are all embedded — so they work fully without a connection.

🧭 How the year is structured

The course runs as a web-first spiral: concepts are revisited with increasing depth rather than "mastered" once. Q1–Q4 form the core lane (everyone ships a real, deployed app). Q5–Q6 are the extension lane for faster students — engineering depth and a second, bigger capstone.

You'll build a shippable artifact every quarter, from a static site (Q1) to an interactive client app (Q2), a full-stack CRUD app (Q3), and finally a deployed group capstone (Q4).

🤖 A note on using AI

AI is a taught skill here, not a secret. You may use AI to generate code only in technologies you have personally unlocked by passing a hands-on "gate." This guarantees you can always verify what the AI produces. See each lecture's activities for which phase applies.
